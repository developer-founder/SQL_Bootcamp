## 🗄️ Структура базы данных

Проект использует учебную базу данных пиццерий со следующей структурой:

### 📊 Таблицы базы данных:

1. **pizzeria** (пиццерии)
   - `id` - первичный ключ
   - `name` - название пиццерии
   - `rating` - средний рейтинг (0-5 баллов)

2. **person** (клиенты)
   - `id` - первичный ключ
   - `name` - имя клиента
   - `age` - возраст
   - `gender` - пол
   - `address` - адрес

3. **menu** (меню)
   - `id` - первичный ключ
   - `pizzeria_id` - внешний ключ на pizzeria
   - `pizza_name` - название пиццы
   - `price` - цена

4. **person_visits** (посещения)
   - `id` - первичный ключ
   - `person_id` - внешний ключ на person
   - `pizzeria_id` - внешний ключ на pizzeria
   - `visit_date` - дата посещения

5. **person_order** (заказы)
   - `id` - первичный ключ
   - `person_id` - внешний ключ на person
   - `menu_id` - внешний ключ на menu
   - `order_date` - дата заказа

## 🛠️ Выполненные задания

### ✅ **Задание 00: Базовые запросы SELECT**

- Написание простого SELECT запроса
- Выборка данных по условию (город Казань)
- Возврат конкретных полей (имя и возраст)

### ✅ **Задание 01: Фильтрация и сортировка**

- Фильтрация по нескольким условиям (город и пол)
- Сортировка результатов по имени
- Работа с условиями WHERE и ORDER BY

### ✅ **Задание 02: Диапазоны значений**

- Использование операторов сравнения (<=, >=)
- Применение ключевого слова BETWEEN
- Сравнение двух подходов к фильтрации диапазонов

### ✅ **Задание 03: Уникальные значения и сложные условия**

- Использование DISTINCT для уникальных значений
- Сложные условия с использованием OR
- Фильтрация по датам и идентификаторам
- Сортировка по убыванию

### ✅ **Задание 04: Форматирование вывода**

- Конкатенация строк и значений
- Форматирование вывода в читаемый вид
- Использование кавычек и специальных символов
- Создание вычисляемых полей

### ✅ **Задание 05: Подзапросы в части SELECT**

- Использование внутренних подзапросов
- Фильтрация по нескольким идентификаторам
- Работа с датами в условиях
- Ограничения на использование JOIN

### ✅ **Задание 06: Условные выражения**

- Добавление вычисляемого столбца с проверкой
- Использование условной логики в SQL
- Сравнение значений в запросах

### ✅ **Задание 07: Интервалы и оператор CASE**

- Применение оператора CASE для категоризации

- Сортировка по вычисляемому полю

### ✅ **Задание 08: Работа с числами**
